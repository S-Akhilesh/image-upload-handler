# Image Upload Handler

A React application for uploading images to Cloudinary with drag-and-drop support, real-time progress tracking, and error handling.

## Features

- **Drag & Drop**: Upload images by dragging and dropping them into the upload area
- **File Browser**: Select images using the file browser
- **Multiple Uploads**: Upload multiple images simultaneously
- **Progress Tracking**: Real-time upload progress for each image
- **Error Handling**: Validation errors and upload retry functionality
- **Image Previews**: Preview selected images before uploading
- **File Validation**: Supports PNG, JPG, GIF formats (max 5MB per file)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- Cloudinary account with upload preset

### Installation

1. Clone the repository and install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the port shown in the terminal).
